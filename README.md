# integracao-mp-sdkjava-
Integration backend with Java SDK Mercado Pago.

## Prepare your environment
- Set the variables ACCESS_TOKEN_TEST and PUBLIC_KEY_TEST in your system variables.
```Ex in Linux or Mac: 
export PUBLIC_KEY_TEST_OK="TEST-09"
export PATH=$PATH:$PUBLIC_KEY_TEST_OK

export ACCESS_TOKEN_TEST_OK="TEST-"
export PATH=$PATH:$ACCESS_TOKEN_TEST_OK

```
